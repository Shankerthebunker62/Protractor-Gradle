# "# Protractor-Gradle" 

<i>Protractor</i> Data-Driven Automation Framework using Excel data sheet for UI Object Repository as well as Test Data. This is a Action-Component based BDD model as Jasmine. For Mobile Application Automation.
</br></br>

# Execution:

* gradle wrapper
* gradle installGulp
* gradle clean build --refresh-dependencies
* gradle task e2e
* gradle wrapper installGulp clean build --refresh-dependencies task e2e --scan
* gradle --stop
</br></br>

# NPM Updates:

* sudo npm install npm-check --global --save
</br></br>
* sudo npm install npm-check-updates --global --save
* sudo npm install npm-check-updates --global --update-all --save
* sudo npm install
</br></br>
